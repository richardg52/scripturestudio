⚡ How to Push an Update Live

After Claude makes any change to the website, open Terminal and run these commands:

1
Navigate to your website folder:
cd "/Users/richard/Desktop/RGC-MASTER/Websites and Designs/Scripture-Studio-2"
2
Push to GitHub (Netlify auto-deploys in ~30 seconds):
git add . && git commit -m "update" && git push
💡 Tip: If Terminal asks for a password, paste your Personal Access Token. Never use your regular GitHub password. Regenerate at GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic).

🔑 GitHub Personal Access Token

If you ever need a new token:

1
Go to github.com → click your profile picture → Settings
2
Scroll to the bottom of the left sidebar → Developer settings
3
Personal access tokens → Tokens (classic) → Generate new token (classic)
4
In the Note field type Scripture Studio Deploy. Check the repo box, set expiration, click Generate. Copy and save it somewhere safe.
5
Run this in Terminal to update the saved token (replace YOUR_TOKEN):
cd "/Users/richard/Desktop/RGC-MASTER/Websites and Designs/Scripture-Studio-2" git remote set-url origin https://richardg52:YOUR_TOKEN@github.com/richardg52/scripture-studio-2.git
